# separador de sílabas

aplicación web sencilla en un solo archivo html para separar una palabra en sílabas y permitir reorganizarlas mediante arrastrar y soltar.

## características

- entrada de texto para escribir una palabra.
- separación básica en sílabas.
- reordenamiento de sílabas con drag & drop.
- vista inmediata de la palabra resultante.

## tecnologías

- html
- css
- javascript

## uso

1. abre el archivo `index.html` en tu navegador.
2. escribe una palabra en el campo de texto.
3. arrastra las sílabas para cambiarlas de orden.
4. la palabra final se actualizará automáticamente.

## estructura del proyecto

- `index.html`: contiene toda la interfaz, estilos y lógica de la aplicación.

## notas

- la separación silábica es básica y puede no cubrir todas las reglas del español.
- el proyecto no requiere instalación ni dependencias.
